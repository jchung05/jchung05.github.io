Here are some of the projects I've done in my free time, programming or not. They showcase a range of different development expertise with a particular focus on mobile and web development.

Excel Schedule Maker
This is a VBA application that allows a user to enter a list of employees, their required numbers of shifts, and the shift schedule into a spreadsheet to generate a work schedule with all the employees in selectable, conditional dropdown lists for each shift. Both schedule generation and active conditional checking occurs in linear time. It allows the user both to select employees for each shift by hand or to hit a button to automatically optimize and fill each shift with the required number of employees.

WEBSITES - These are some of the sites I've worked on.

Rich's Smog & Repair, Fremont, CA Jan 2015 – Current
Rich's Smog & Repair is my uncle's automotive repair shop located near the Tesla factory in Fremont, CA. I am the webmaster and currently handle the migration from Apache 2.2 to 2.4 and have been updating the site to include interactive PHP and AJAX as well as all upkeep for future content. ThriveSync, Brooklyn, NY Jan 2016 – Current
ThriveSync is a mental health modeling application by ThriveStreams. I offered my free time to moonlight on the project to support my colleague who had flown over to the east coast to c hase his dream of a startup. I am in charge of refactoring the code base from using static strings to a centralized YAML file which utilizes i18n in Rails. I also made a RESTful language selector for the site using parameter passing to change the default language. The team has successfully implemented simple Han, Spanish, and English. The site is currently shutting down.
